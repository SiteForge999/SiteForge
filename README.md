# SiteForge — Конструктор Landing Page «Под Ключ» (Telderi Edition)

Добро пожаловать в проект **SiteForge** — премиальный, современный block-based конструктор сайтов и посадочных страниц. Этот проект создан без внешних тяжелых зависимостей (чистый HTML, CSS и JavaScript), что обеспечивает максимальную скорость работы (Score 100 в PageSpeed) и абсолютную простоту в поддержке.

## 🚀 Почему этот проект имеет высокую ценность для продажи:
1. **100% Бесплатный стек**: Проект разворачивается и работает полностью бесплатно. Покупателю не нужно платить за хостинг или серверы.
2. **Передача аккаунта «Под ключ»**: Вместе с кодом покупатель получает выделенный Google-аккаунт, на который зарегистрированы все связанные сервисы (GitHub, Vercel, Supabase).
3. **Функционал экспорта**: Конструктор экспортирует чистейший, SEO-оптимизированный HTML-код с инлайновыми стилями блоков, который ваши клиенты могут скачать в 1 клик и запустить на любом хостинге.
4. **Облачное сохранение**: Наличие интеграции с Supabase позволяет сохранять проекты пользователей прямо в базу данных, делая из этого полноценный SaaS.

---

## 🛠️ Архитектура и технологии
* **Frontend**: HTML5, Ванильный CSS3 (модный стек с эффектом Glassmorphism, адаптивная верстка, поддержка светлых/темных тем в блоках).
* **JS-Engine**: Чистый JavaScript (`app.js`), реализующий реактивное обновление интерфейса, редактирование текста прямо на холсте (`contenteditable`), изменение стилей блоков и сборщик/экспорт кода.
* **Backend**: **Supabase** (PostgreSQL база данных) для хранения проектов.
* **Хостинг**: **Vercel** (быстрый деплой из репозитория GitHub за пару кликов).

## 📋 Инструкция по развертыванию (для нового владельца)

Благодаря передаче Google-аккаунта, проект уже настроен. Но если вы захотите развернуть его с нуля или привязать свои базы данных, выполните следующие шаги:

### 1. Настройка Базы Данных (Supabase)
В панели управления Supabase создайте таблицу `projects`. Для этого перейдите в раздел **SQL Editor** и выполните следующий запрос:

```sql
CREATE TABLE projects (
  id TEXT PRIMARY KEY,                   -- Уникальный идентификатор проекта
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL,
  title TEXT NOT NULL,                   -- SEO заголовок страницы
  description TEXT,                      -- SEO описание страницы
  font_headings TEXT DEFAULT 'Outfit',   -- Шрифт заголовков
  font_body TEXT DEFAULT 'Inter',        -- Шрифт основного текста
  blocks JSONB NOT NULL                  -- JSON массив с блоками и их содержимым
);

-- Разрешаем публичный доступ к таблице (без авторизации):
CREATE POLICY "Enable access for everyone" ON projects FOR ALL USING (true) WITH CHECK (true);
ALTER TABLE projects ENABLE ROW LEVEL SECURITY;
```

### 2. Подключение к сайту
При запуске сайта нажмите на кнопку **«Cloud Save»** (Облачное сохранение) в шапке конструктора. Введите:
* **Supabase Project URL** (из настроек проекта API -> Project URL)
* **Supabase Anon Key** (из настроек проекта API -> anon / public key)

Нажмите **«Connect Database»**. Данные будут сохранены в браузере, и теперь при нажатии «Save Project» конфигурация страницы будет улетать в облако, а при нажатии «Load Project» — загружаться обратно.

### 3. Публикация на Vercel
1. Создайте репозиторий на вашем GitHub (входит в передаваемый аккаунт) и загрузите туда файлы:
   * `index.html`
   * `style.css`
   * `app.js`
   * `supabase.js`
   * `README.md`
2. Перейдите на Vercel, авторизуйтесь через переданный GitHub-аккаунт.
3. Нажмите **Add New** -> **Project**, выберите ваш репозиторий `site-forge` и нажмите **Deploy**.
4. Сайт будет опубликован и доступен по бесплатному SSL-адресу (например, `site-forge.vercel.app`).

## 🔑 Процедура передачи прав (для передачи лота на Telderi)

Для безопасной передачи проекта покупателю выполните следующие шаги:
1. **Передача Google-аккаунта**: Передайте логин и пароль от выделенного Google-аккаунта.
2. **Настройки безопасности**: Покупатель должен сразу зайти в аккаунт, изменить пароль, привязать свой номер телефона и изменить резервный адрес электронной почты.
3. **Проверка GitHub, Vercel и Supabase**: Все эти сервисы привязаны к этой почте. Войдя в них через Google-авторизацию или почту, покупатель получает полный контроль над кодом, хостингом и базой данных без необходимости осуществлять перенос инфраструктуры.
